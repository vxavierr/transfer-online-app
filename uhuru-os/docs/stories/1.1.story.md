# Story 1.1 — Monorepo Scaffold and Base Configuration

**Epic:** 1 — Foundation, Auth and Project Setup
**Status:** Done
**Points:** 5

## Description

As a developer, I want a Turborepo monorepo with apps/web (React + Vite + Tailwind 4), apps/api (Bun + Hono), and packages/shared scaffolded, so that I have a working development environment with build, lint, and typecheck commands.

## Acceptance Criteria

1. Monorepo inicializado com Turborepo, usando Bun como package manager root
2. `apps/web` configurado com React 19 + Vite + TypeScript + Tailwind 4; pagina inicial exibe "Hello Uhuru OS"
3. `apps/api` configurado com Bun + Hono + TypeScript; `GET /health` retorna `{ status: "ok", version: "0.1.0" }`
4. `packages/shared` criado com export de tipos basicos: `ApiResponse<T>`, `ApiError`, `Pagination`
5. `packages/agent-harness` criado como scaffold vazio (package.json + tsconfig.json + `src/index.ts` vazio com export placeholder)
6. `turbo build`, `turbo lint`, `turbo typecheck` executam sem erros em todos os packages
7. ESLint + Prettier configurados no root com configs compartilhadas via `eslint.config.js` e `.prettierrc`
8. `.env.example` documentando todas as variaveis de ambiente necessarias ao projeto

## Scope

### IN
- Estrutura de diretorios do monorepo conforme DESIGN-DOC secao 2
- Configuracao do Turborepo pipeline (build, lint, typecheck, test, dev)
- Configuracao base de cada app/package (package.json, tsconfig.json, vite.config.ts, etc.)
- Scripts de dev: `bun run dev` no root sobe web e api simultaneamente
- `.gitignore`, `bun.lockb`, `turbo.json`, `tsconfig.base.json` no root

### OUT
- Nenhuma feature de negocio (auth, clientes, etc.)
- Docker / containerizacao
- CI/CD pipeline
- Supabase setup (Story 1.2)
- Componentes de UI alem do placeholder

## Technical Notes

### Stack Exata
- Package manager: Bun (nao npm/yarn)
- Turborepo pipeline conforme `turbo.json` do DESIGN-DOC secao 2.2
- React 19 com Vite 6+ e `@vitejs/plugin-react`
- Tailwind 4 (usa `@import "tailwindcss"` no CSS, nao `tailwind.config.js`)
- Hono + `@hono/node-server` rodando no Bun
- TypeScript 5.x com `strict: true`, paths configurados via `tsconfig.base.json`

### Estrutura de Diretorios
```

├── apps/web/
├── apps/api/
```
