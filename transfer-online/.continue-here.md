# BLOCKING CONSTRAINTS — Read Before Anything Else

- [ ] **CONSTRAINT: Android intocável** — Qualquer mudança em código iOS DEVE ser condicional a `Capacitor.getPlatform() === 'ios'`. `app/android/**` e `TelemetryForegroundBridge.js` PROIBIDOS. @qa valida via `git diff --stat -- 'transfer-online/app/android/'` (deve ser vazio).
- [ ] **CONSTRAINT: registerPlugin retorna proxy truthy** — `Capacitor.registerPlugin('X')` SEMPRE retorna objeto truthy no nativo, mesmo sem implementação para a plataforma atual. NUNCA usar `!plugin` para detectar plataforma. Sempre `Capacitor.getPlatform()`.
- [ ] **CONSTRAINT: QA PASS ≠ runtime OK** — Story 1.1 teve QA PASS em todos os AC mas o código nunca rodou no iOS por bug de seletor. QA deve declarar explicitamente se valida runtime ou só código.

- [ ] **CONSTRAINT: Nunca `git add -A`** — Há muitos arquivos modificados pré-existentes no monorepo (Android FGS, uhuru-os). Sempre adicionar por path específico.
