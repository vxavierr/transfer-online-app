# Story 7.2 — Data Import — CSV Upload and Google Sheets

**Epic:** 7 — API Connector and Data Import
**Status:** Draft
**Points:** 8

## Description

As a user, I want to import data from CSV files and Google Sheets into the platform, so that I can onboard historical data and external sources.

## Acceptance Criteria

1. Tabela `import_jobs` criada via migration: `id` (uuid PK), `source_type` (text NOT NULL CHECK IN ('csv','google_sheets')), `source_reference` (text NOT NULL — file path no Storage ou Sheet URL), `target_table` (text NOT NULL), `column_mapping` (jsonb NOT NULL — `{ csv_col: db_col }`), `status` (text NOT NULL CHECK IN ('pending','processing','completed','failed') DEFAULT 'pending'), `rows_imported` (integer NOT NULL DEFAULT 0), `errors` (jsonb NULLABLE — array de erros por linha), `created_by` (uuid NOT NULL FK profiles(id)), `created_at` (timestamptz)
2. Upload CSV: `POST /api/v1/import/csv` aceita `multipart/form-data` com arquivo CSV; detecta colunas automaticamente; retorna job_id e colunas detectadas para mapeamento
3. Preview: `POST /api/v1/import/:id/preview` retorna primeiras 5 linhas com mapeamento atual aplicado (mostra como os dados ficarao no sistema)
4. Execucao: `POST /api/v1/import/:id/execute` processa o job; linhas com erro sao logadas em `errors` mas nao bloqueiam importacao das demais linhas validas
5. Google Sheets: `POST /api/v1/import/google-sheets` aceita URL da Sheet; requer OAuth do Google (reutiliza credenciais OAuth da Story 3.2 ou Google API key configurada no servidor); retorna colunas detectadas igual ao CSV
6. UI: pagina de importacao com stepper (3 passos: Selecionar fonte → Mapear colunas → Confirmar e importar); historico de importacoes com status, linhas importadas, erros

## Scope

### IN
- Migration `import_jobs` com RLS
- Bucket Storage `import-uploads` para CSV temporarios
- Service `import.service.ts` com logica de parse CSV, Google Sheets fetch, mapeamento e upsert
- Rotas Hono de importacao
- UI stepper de 3 passos e historico

### OUT
- Importacao de campanhas de ads via CSV (metricas vem da API — nao de CSV no MVP)
- Scheduled/recurring imports
- Importacao em background async (sincrono no MVP — processo durante a request; para arquivos grandes, pos-MVP)
- Deltas/merge inteligente (insert only no MVP)
