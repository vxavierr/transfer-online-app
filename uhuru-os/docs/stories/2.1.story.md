# Story 2.1 — Application Layout — Sidebar, Topbar and Navigation

**Epic:** 2 — Client Management and Base UI
**Status:** Draft
**Points:** 5

## Description

As a user, I want a consistent application layout with sidebar navigation, topbar with user info, and routing between modules, so that I can navigate the platform intuitively.

## Acceptance Criteria

1. Layout base com `AppShell` contendo: sidebar colapsavel (260px expandida, 64px colapsada), topbar (altura 56px com nome do usuario, avatar e botao de logout) e area de conteudo principal
2. Sidebar com itens de navegacao: Dashboard, Clientes, Campanhas, Tarefas, Automacoes, API Connector, Importacao; cada item com icone e label; item ativo destacado visualmente
3. React Router v6 configurado com rotas para todos os modulos (paginas placeholder com titulo e descricao do modulo)
4. Navegacao funcional entre todas as rotas sem reload de pagina
5. Sidebar colapsa em telas menores (abaixo de 1024px) com toggle via botao no topbar
6. Componentes base do shadcn/ui instalados e customizados: Button, Input, Card, Table, Dialog, Sheet, Tabs, Badge, Avatar, Toast (Sonner), Separator, ScrollArea
7. Tema dark como default; estrutura pronta para toggle dark/light (Story futura)
8. `ThemeProvider` configurado com `next-themes` ou equivalente para Vite

## Scope

### IN
- Componentes: `AppShell`, `Sidebar`, `Topbar`, `SidebarItem`
- Configuracao do React Router com todas as rotas de alto nivel
- Instalacao e configuracao do shadcn/ui com tema Uhuru (cores da identidade Uhuru)
- Paginas placeholder para cada modulo (apenas layout + heading)
- Estado de colapso do sidebar (persistido em localStorage)

### OUT
- Conteudo real de qualquer pagina (vem nas stories de cada modulo)
- Chat do agente (Epic 8)
- Responsividade mobile completa (pos-MVP)
- Design tokens finais da Uhuru (aguarda @ux-design-expert)

### Rotas
```typescript

  { path: '/dashboard', element: <DashboardPage /> },
  { path: '/clients', element: <ClientsPage /> },
  { path: '/clients/:id', element: <ClientDetailPage /> },
  { path: '/campaigns', element: <CampaignsPage /> },
  { path: '/tasks', element: <TasksPage /> },
  { path: '/automations', element: <AutomationsPage /> },
  { path: '/api-connectors', element: <ApiConnectorPage /> },
```
